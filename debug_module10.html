模块10调试检查

请在浏览器控制台运行以下JavaScript代码：

// 检查模块10导航是否存在
console.log('检查模块10导航元素:');
const tenthNav = document.querySelector('[data-view="tenthModule"]');
console.log('模块10导航元素:', tenthNav);

// 检查模块10视图是否存在  
const tenthView = document.getElementById('tenthModuleView');
console.log('模块10视图元素:', tenthView);

// 检查switchToTenthModule函数是否存在
console.log('switchToTenthModule函数:', typeof switchToTenthModule);

// 如果导航元素存在但不可见，手动调用切换函数
if (typeof switchToTenthModule === 'function') {
    console.log('手动切换到模块10...');
    switchToTenthModule();
} else {
    console.log('switchToTenthModule函数未定义');
}

// 检查模块10 API是否可用
fetch('/api/eye-index/available-configs')
    .then(response => response.json())
    .then(data => console.log('模块10 API响应:', data))
    .catch(error => console.error('模块10 API错误:', error));

如果上述检查失败，请尝试：

  1. 完全关闭浏览器
  2. 重新启动服务器: python start_server.py
  3. 重新打开浏览器访问 http://localhost:8080
